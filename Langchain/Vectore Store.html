2 – Loading the document

Goal

Undestand how to work with documents in Langchain

1. Langchain `Document` class

It is LangChain’s canonical data container for text.

<class 'langchain_core.documents.base.Document'>

Think of it as a minimal, standardized record that flows through the entire pipeline:

Loader → Splitter → Embeddings → VectorStore → Retriever → LLM

Every stage expects Document objects.

What a Document actually contains

At minimum:

Document(
page_content: str,
metadata: dict
)


Example:


Document(
page_content="Memcached is a distributed in-memory cache...",
metadata={
"source": "documents/system_design.pdf",
"page": 12
}
)

2. Document Loader Class

A Document Loader class in LangChain is a source adapter.

Its only responsibility is to turn raw data (files, URLs, APIs, DB rows, etc.) into Document objects.

Formal definition

A Document Loader is a class that implements a load() method and returns
List[langchain_core.documents.base.Document].

What a Document Loader MUST do

  • Reads from a source
File system

- Web
- PDF
- Database
- API
  • Extracts raw text
  • Wraps text into `Document` objects
  • Adds metadata

Example: PyPDFLoader

from langchain_community.document_loaders import PyPDFLoader
loader = PyPDFLoader("file.pdf")
docs = loader.load()
Output:

[
Document(
page_content="Text from page 1",
metadata={"source": "file.pdf", "page": 1}
),
Document(
page_content="Text from page 2",
metadata={"source": "file.pdf", "page": 2}
)
]

Every page become a document

3. Directory Loader

DirectoryLoader is a filesystem orchestrator.

It does not parse content.

It only walks directories and delegates each file to a real Document Loader.

What DirectoryLoader does (step-by-step)

  • 1. Scan directory recursively
  • 2. Match files using glob
  • 3. For each file: (1) instantiate loader_cls(file_path), (2) call loader.load()
  • 4. Collect all Document objects
  • 5. Return List[Document]
from langchain_community.document_loaders import DirectoryLoader, PyPDFLoader

loader = DirectoryLoader(
    path="./documents",
    glob="**/*.pdf",
    loader_cls=PyPDFLoader
)

documents = loader.load() # returns a list of LangchainDocuments

1. ingestion.py

import os

from dotenv import load_dotenv
from langchain_text_splitters import RecursiveCharacterTextSplitter
from langchain_community.document_loaders import DirectoryLoader, PyPDFLoader
from langchain_chroma import Chroma
from langchain_openai import OpenAIEmbeddings

load_dotenv()

# loader and load the documents
documents_dir = "./documents"
if not(os.path.exists(documents_dir)) or  not os.listdir(documents_dir):
    print(f"Warning: Empty or no document {documents_dir} dir")
loader = DirectoryLoader(
    documents_dir,
    glob = "**/*.pdf",
    loader_cls=PyPDFLoader,
    loader_kwargs = {"mode":"single"}
)
documents_list = loader.load()

# splitter and get the document splits
text_splitter = RecursiveCharacterTextSplitter.from_tiktoken_encoder(
    chunk_size = 90,
    chunk_overlap = 10,
)

documents_chunk = text_splitter.split_documents(documents_list)

# Vectore store constants
collection_name = "rag_chroma"
persist_directory = "./.chroma"

# create vectore store
vector_store = Chroma.from_documents(
    documents = documents_chunk,
    collection_name = collection_name,
    persist_directory= persist_directory,
    embedding = OpenAIEmbeddings(),
)

2. retriever.py

from langchain_chroma import Chroma
from langchain_openai import OpenAIEmbeddings
from dotenv import load_dotenv

load_dotenv()

# Vectore store constants
collection_name = "rag_chroma"
persist_directory = "./.chroma"


# create the retriever
retriever    = Chroma(
    collection_name = collection_name,
    persist_directory = persist_directory,
    embedding_function= OpenAIEmbeddings()
).as_retriever()


if __name__ == "__main__":
    result = retriever.invoke(input="What is memcache",k=10)
    printed = False
    for val in result:
        if not printed:
            print(type(val))
            print()
            printed= True
        print(val.page_content)
        print()

3. main.py

from dotenv import load_dotenv
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

from retriever import retriever

load_dotenv()

prompt_template = """ 
You are an assistant for question-answering tasks. Use the following pieces of retrieved context to answer the question. If you don't know the answer, just say that you don't know. Use three sentences maximum and keep the answer concise.
Question: {question} 
Context: {context} 
Answer:
"""

prompt = ChatPromptTemplate.from_template(prompt_template)

llm = ChatOpenAI()

chain = prompt | llm | StrOutputParser()


if __name__ == "__main__":
    user_question = input("You: ")
    retrieved_documents = retriever.invoke(user_question,k=5)
    context = [doc.page_content for doc in retrieved_documents]
    answer = chain.invoke({
        "question" : user_question,
        "context": context,
    })
    print("\n[AI] ",answer)
    print()
    print("#"*30)
    print("\nRetrived documents")
    for txt in context:
        print(txt)
        print()
